Descripción profesional

Santiago Moya
Estudiante de Ingeniería en Tecnologías de la Información y Comunicaciones con enfoque en Ingeniería de Software y Sistemas Electrónicos.

Actualmente soy un estudiante con fuerte orientación práctica, especializado en el desarrollo de software, bases de datos y proyectos tecnológicos integrales. Mi formación académica me ha permitido combinar programación, electrónica y gestión de proyectos, desarrollando soluciones funcionales tanto en entornos digitales como físicos.

He trabajado en proyectos académicos y prototipos tecnológicos que integran programación en Java, desarrollo web (HTML, CSS), bases de datos MySQL y sistemas electrónicos con microcontroladores como ESP32. Me enfoco en la construcción de soluciones escalables, organizadas y bien documentadas, cumpliendo con estándares académicos y técnicos.

Áreas de especialización

  • Desarrollo de software (Java, estructuras de datos, validaciones con pilas)
  • Bases de datos relacionales (MySQL)
  • Desarrollo web básico (HTML, CSS)
  • Electrónica y prototipado (ESP32, sensores, BJTs, circuitos)
  • Modelado y documentación de proyectos (diagramas, Gantt, UML)

Moya Garage

Es un sistema de gestión para un lote de compra y venta de automóviles. El proyecto integra control de inventario, gestión de clientes y administración de ventas, desarrollado como solución tecnológica académica con visión empresarial real.
El sistema busca optimizar procesos internos, mejorar el control administrativo y facilitar la toma de decisiones mediante almacenamiento estructurado de datos.

Tecnologías utilizadas

  • Java (Programación orientada a objetos)
  • MySQL (Base de datos relacional)
  • Visual Studio 2022
  • Modelado
  • Diagramas entidad-relación
  • Metodología estructurada de desarrollo

Apple Second Life

Es un marketplace especializado en la compra, reacondicionamiento y venta de productos Apple de segunda mano. El modelo de negocio es B2C, donde la empresa adquiere dispositivos usados, los certifica y los vende directamente al consumidor final.

El proyecto busca ofrecer confianza, garantía y experiencia premium, diferenciándose de la reventa informal mediante control de calidad, pagos seguros y logística confiable.

Tecnologías utilizadas

  • Plataforma web (HTML, CSS).
  • Base de datos para control de inventario.
  • Sistema de pagos seguros.
  • Panel administrativo para gestión de productos.
